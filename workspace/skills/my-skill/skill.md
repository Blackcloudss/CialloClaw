# my-skill

提供文件处理和分析的工具。

## Tools

### process-file
**命令**: python scripts/process.py --input {{input}} --output {{output}}
**描述**: 处理输入文件并输出结果

**参数**:
- input (string): 输入文件路径
- output (string): 输出文件路径

### analyze
**命令**: ./scripts/analyze.sh {{path}}
**描述**: 分析指定路径的数据

**参数**:
- path (string): 要分析的目录路径
