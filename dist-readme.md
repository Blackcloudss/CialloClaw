# CialloClaw EXE Quick Start

1. Run `CialloClaw.exe`
2. The app auto-starts bundled `core\go-core.exe` when needed
3. No extra terminal commands required for end users

Troubleshooting:
- If port 18080 is occupied, close conflicting service and restart CialloClaw.exe
- If first run is blocked by Windows SmartScreen, choose "More info" -> "Run anyway"
